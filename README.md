# 🚀 Job Application Tracker

A modern web application that helps job seekers organize, track, and analyze their job applications in one place.

---

## 📌 Project Overview

The Job Application Tracker is designed to simplify the job search process by allowing users to manage applications, monitor progress, and gain insights through analytics.

This project evolved through multiple phases, focusing on UI/UX improvements, feature enhancements, and full-stack integration using modern technologies.

---

## ✨ Features

### 📱 UI/UX Improvements
- Fully responsive design (mobile, tablet, desktop)
- Improved dashboard layout and spacing
- Clean empty states with actionable messages
- Modern dashboard cards with icons and hover effects

### 🏷 Application Management
- Add, edit, and manage job applications
- Track application status:
  - Applied
  - Interview
  - Offer
  - Rejected
- Color-coded status badges for better visibility

### 📊 Analytics Dashboard
- Applications by Status (Pie Chart)
- Applications Over Time (Line Chart)
- Applications by Company (Bar Chart)

### 🔎 Productivity Features
- Search applications by job title
- Filter by status and company
- Store job posting links
- Track resume versions used per application

### 🔐 Authentication & Database
- User authentication using Supabase
- Secure data storage with PostgreSQL
- User-specific data access with Row Level Security (RLS)

### 📤 Export Feature
- Download job application data as CSV

---

## 🛠 Tech Stack

- **Frontend:** Next.js, React
- **Styling:** Tailwind CSS
- **Backend / Database:** Supabase (PostgreSQL)
- **Charts:** Chart.js
- **Deployment:** Vercel
